## removeDuplicates

```js client
let array = [1, 2, 2, 2, 3];
let filter = removeDuplicates(array);
console.log(filter); // outPut [1,2,3]
```

## TurnOffLogs

```js client
console.log("Before TurnOff Logs"); // print normally

TurnOffLogs();

console.log("After TurnOff Logs"); // that log will not print
```
